PDF合并工具 v1.0
====================

📋 软件介绍
-----------
这是一个PDF分页合并工具，可以将多个PDF文件按页面生成独立的合并文件。
主要功能：
- 将多个PDF按页面交替合并（第1页：A1+B1+C1，第2页：A2+B2+C2...）
- 自动从第三个PDF文件中提取申请人姓名作为输出文件名
- 支持OCR文字识别，可处理扫描版PDF
- 内置Tesseract OCR引擎和Poppler工具

🚀 使用方法
-----------
1. 下载编译好的exe文件（见下方获取方式）
2. 将需要处理的PDF文件放入"PDF插入\原始文件"目录
3. 双击运行"运行PDF合并工具.bat"
4. 程序会自动处理并将结果保存到"PDF插入\最终文件"目录

📥 获取exe文件
--------------
由于GitHub文件大小限制，编译后的exe文件需要单独获取：
1. 方式一：从GitHub Releases页面下载
2. 方式二：自行编译（运行：pyinstaller PDF合并工具.spec）
3. 方式三：联系开发者获取

📝 文件说明
-----------
- PDF合并工具.exe：主程序文件（需单独获取）
- 运行PDF合并工具.bat：启动脚本（推荐使用）
- PDF插入\原始文件\：放置待处理PDF文件的目录
- PDF插入\最终文件\：输出结果文件的目录
- 使用说明.txt：详细使用说明
- README.md：技术文档

⚠️ 注意事项
-----------
1. 请确保PDF文件按正确顺序命名（如：1项目.pdf, 2项目.pdf, 3项目.pdf）
2. 第三个PDF文件用于提取申请人姓名，请确保包含"申请人："字段
3. 程序需要OCR功能，已内置相关组件，无需额外安装
4. 首次运行可能需要较长时间，请耐心等待
5. 如遇到问题，请查看控制台输出的错误信息

🔧 系统要求
-----------
- Windows 10/11 (64位)
- 至少2GB可用内存
- 足够的磁盘空间存储输出文件
